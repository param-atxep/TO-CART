## Features
- User Authentication (JWT / Firebase / OAuth)
- Product Listings, Categories, Search, Filters
- Cart, Checkout, Coupons, Order Management
- Payment Integration (Stripe/Razorpay mock)
- Wishlist, Reviews, Ratings
- Admin Panel (Dashboard, Products, Orders, Users)
- Responsive Design, Dark Mode, Toasts, Error Handling
